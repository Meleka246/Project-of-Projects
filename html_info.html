Here, we will be talking about what HTML5 is, and where/why we use it.

What is HTML and why is it important?

Html is hyper text markup language, or, what you normally would see in a webpage in 1997.

The reason we use it, is for multiple reasons:

  • Used as a foundation for a webpage
  • Makes for a great beginner language
  • Can be used to create your own webpage(such as here)
  • More simpler to use than other languages that are the same

Where did it come from?

It came from Berners-Lee, at CERN using SGML, an earlier form of language, similar to HTML. It was formally defined as that by the Internet Engineering Task Force (IETF) within the middle of 1993.

What is CSS then?

CSS is an extention to HTML: it is responsible for all the visual elements that the HTML page produces.

Why is it so important?

Nowadays, everyone want there website to look modern, and clean. So, in order to do that without HTML code changing drasticaly, they instead, made CSS, so that HTML would not be too crowded.

What does it stand for?

I stands for Cascading Style Sheets, or in simpler terms, an element used to make things look better.

How is it used, write CSS?

Of course not! You write css similar to how you write HTML

Where did it come from?

When the international organization that owned HTML, thought that the page looked too dull, they tought of a briliant plan to develop a sub-language, thats only purpose was to make pages look better.
